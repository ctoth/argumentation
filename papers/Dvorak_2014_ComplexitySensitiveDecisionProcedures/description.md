Presents complexity-sensitive decision procedures for hard abstract-argumentation reasoning tasks under preferred, semi-stable, and stage semantics.
It contributes fragment and distance-based complexity results, then instantiates them as a SAT-based CEGAR-style framework implemented in CEGARTIX.
It is relevant to this project because it gives concrete base-semantics encodings, oracle-call structure, and benchmark evidence for incremental SAT-backed argumentation reasoning.
