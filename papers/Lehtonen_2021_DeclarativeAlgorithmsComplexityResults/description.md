Presents direct ASP encodings for ABA and ABA+ reasoning in the flat logic-programming fragment, avoiding translation through abstract argumentation frameworks.
Establishes complexity results showing that ABA+ reverse attacks induced by preferences raise several verification and acceptance tasks, while stable-semantics acceptance remains at the ABA complexity level.
Relevant to this project's backend because it gives solver-oriented representations, polynomial subchecks, oracle-level complexity expectations, and benchmark evidence for ABA/ABA+ implementation choices.
