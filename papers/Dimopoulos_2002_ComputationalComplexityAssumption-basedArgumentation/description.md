Classifies the complexity of credulous and sceptical reasoning for ABA instances of theorist, circumscription, logic programming, default logic, and autoepistemic logic under admissibility and preferability semantics. Shows that the new semantics often match or exceed stable-extension complexity, reaching `Pi^p_4`-completeness for AEL sceptical preferred reasoning. Relevant to this project because it gives solver-capability boundaries for ABA/default-reasoning semantics and warns that admissible/preferred reasoning is not automatically a cheaper implementation target.
