A unified framework for ABA dispute derivations that parameterizes GB-, AB-, and IB-dispute procedures as instances of X-dispute derivations. It proves soundness and completeness for grounded, admissible, ideal, and related semantics, and extends the framework to structured X-dispute derivations that explicitly compute arguments and attacks. It is directly relevant to implementing a single reusable ABA proof engine that can emit both assumption supports and provenance-bearing dialectical structures.
