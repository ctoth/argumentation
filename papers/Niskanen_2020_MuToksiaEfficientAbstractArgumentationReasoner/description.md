Describes mu-toksia, the C++ SAT-based abstract-argumentation reasoner that won all main-track reasoning tasks at ICCMA 2019. Its central architectural choice is instantiating a single SAT solver once per execution and driving every reasoning task (DC, DS, SE, EE) through incremental solve-under-assumptions calls against one persistent clause database. It gives explicit SAT encodings of conflict-free, admissible, complete and stable sets, and reduces the harder tasks to reused primitives: grounded via unit propagation, DS-CO via grounded, stable primed by the grounded lower bound, and DS-PR / semi-stable / stage via the Dvořák et al. CEGAR loop over complete labellings (range variables added for semi-stable/stage). Dynamic AFs are covered by conditioning each toggleable attack on a selector variable used as a SAT assumption, so the same static machinery handles attack additions/deletions incrementally. It is the reference "single incremental SAT solver" precedent for this project's task-directed af_sat backend.
