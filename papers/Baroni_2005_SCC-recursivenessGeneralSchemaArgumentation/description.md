Introduces SCC-recursiveness, a general schema that computes any Dung admissibility-based semantics by decomposing the defeat graph into strongly connected components, processing them in topological order, and applying a semantics-specific base function to each single-SCC sub-framework while injecting parent SCCs' choices as a defeated/defended/provisionally-defeated partition. It proves stable, complete, preferred, and grounded are all SCC-recursive (each with an explicit base function), shows weak base-function constraints guarantee conflict-freeness and grounded agreement, and defines four new semantics (AD1, AD2, CF1, CF2) that repair preferred semantics' asymmetric cycle handling. It is the theoretical foundation for this project's core/scc_recursive.py and the direct precedent for wiring query-directed SCC decomposition into SAT-based acceptance solving.
